# AutoFlow Pro — Frontend

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Configure backend URL
cp .env.example .env
# Edit .env: set VITE_API_URL to your backend URL

# 3. Start development server
npm run dev
# → http://localhost:5173

# 4. Build for production
npm run build
# → dist/ folder ready to serve
```

## Requirements

- Node.js ≥ 18
- AutoFlow Pro Backend running (see backend ZIP)

## Features

- 22 React components
- 14 social media platforms
- Dark / Light theme
- EN / हिन्दी i18n
- PWA (installable, offline shell)
- Web Push notifications
- Real-time via Socket.io
- Mobile responsive (3 breakpoints)
- Visual Flow Builder
- GA4 + Meta Pixel tracking
